SOS Emergency System

This project is a portable safety device designed to provide immediate assistance in an emergency. Using an Arduino, a GPS module, and a GSM cellular module, it functions as a two-button panic system.

Button 1 (Location): Instantly grabs real-time coordinates and texts a map link.
Button 2 (Call): Dials emergency contact and opens the line to listen in.
